# jarvis

Discord Bot in GoLang

**Requirements:**

- Go 1.21
- Make
- Docker

## How to

To run the application use this command:

```shell
make run
```

or

```shell
go run cmd/jarvis.go
```

To build the docker image, use this command:

```shell
make build
```

To test the docker image:

```shell
make test
```
